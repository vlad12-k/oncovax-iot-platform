# Node-RED Flows Artifacts

This directory stores exported Node-RED flows used by the OncoVax demo environment.

## Structure

- `nodered/` — exported flow JSON files and related static artifacts.

## Conventions

- Use descriptive file names, e.g. `oncovax-main-flow.v1.json`.
- Do not commit credentials embedded in flow configuration.
- Prefer environment variables for any endpoint/token references.

## Phase A status

Placeholder structure has been created. Runtime behavior is unchanged.
