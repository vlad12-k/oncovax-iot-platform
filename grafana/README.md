# Grafana Artifacts

This directory stores exported Grafana dashboard JSON for import into Grafana instances.

## Structure

- `dashboards/` — exported dashboard JSON files.

## Conventions

- Use stable names, e.g. `oncovax-operations.v1.json`.
- Keep dashboard exports free from secrets.
- Use environment-level data source configuration after import.

## Phase A status

Placeholder structure has been created for future dashboard exports.
